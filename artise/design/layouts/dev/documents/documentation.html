The navigation

The navigation contains "pages" which represent different levels of navigations. All pages are stored as separate <div> tags in the #navigation tag.

The navigation object switches focus from one element to another, also relocating the cursor. The focus changes when:

  • The user executes action on a folder, having to switch the focus from the current listing to the new, child listing.
  • The user executes back, going one step back in the navigation.

When the focus changes...

  • ... by entering to a new level: The previous item gets it's 'focus' class removed. All new level listings come with the 'focus' class already set on them.
  • ... by going back to a previous level: The current item is removed from the DOM, and the last children of the navigation gets it's 'focus' class back on.
